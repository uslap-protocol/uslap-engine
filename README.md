# USLaP Core

## Quick Start
1. Double-click launch.command
2. Or in Terminal: cd uslap_core
3. Generate app: python3 generator.py 'App Name'

## Example
python3 generator.py 'Surgical Robot' 1 2 3

This creates: projects/Surgical_Robot.md
