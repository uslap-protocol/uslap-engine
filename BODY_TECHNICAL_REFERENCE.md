## 3. QUF GATES (Adapted Body QUF)

All H1-H5 tables (except body_cross_refs, formula_cross_refs, nafs_cross_refs, sensory_cross_refs) carry QUF columns:

| Gate | Values | Description |
|------|--------|-------------|
| q_gate | Q-DIRECT / Q-DERIVED | Q-DIRECT = specific āyah reference. Q-DERIVED = observation about Allah's creation. |
| u_gate | PASS / CONDITIONAL / N/A | PASS = universally applicable. CONDITIONAL = context-dependent. N/A = structural/anatomical. |
| f_gate | PASS / STRUCTURAL / PENDING | PASS = testable prediction. STRUCTURAL = definitional. PENDING = needs falsifiable criteria. |

**H4 QUF: 70/70 (100%)** — all H4 entries have Q-DIRECT attestation + PASS on U and F gates.
